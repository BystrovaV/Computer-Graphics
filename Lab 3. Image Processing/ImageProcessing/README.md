# Алгоритмы и методы обработки изображений
## Быстрова Вероника, 3 курс 13 группа

## Описание
Данное приложение реализует следующие методы обработки:
- эквализация гистограмм в пространстве RGB и HSV;
- линейное контрастирование; 
- сегментация изображений (обнаружение точек, линий и границ);

## Гистограммы. Эквализация гистограммы
Сравним подходы к выравниванию гистограмм

### Линейное контрастирование
Линейное контрастирование будет работать хорошо только на тех изображениях,  
где отсутсвует либо светлые тона, либо темные

### Эквализация гистограммы
Гистограммы можно эквализировать двумя подходами:
- в цветовом пространстве RGB, где выравнивается каждый цветовой канал
- в цветовом пространстве HSV, где выравнивается только яркость

От способа зависит результат эквализации. Если использовать первый подход,  
то может значительно поменяться цветовая палитра изображения, 2-ой способ в этом плане более безопасный,
хотя также может выдавать неожиданные результаты

## Документация. Используемые библиотеки

```
pip install opencv-python

```

## Build
Для сборки в exe файл нужно дополнительно установить библиотеку `PyInstaller`
```
pip install pyinstaller
```
Для компиляции использовать следующую команду
```
pyinstaller --onefile -w main.py
```
